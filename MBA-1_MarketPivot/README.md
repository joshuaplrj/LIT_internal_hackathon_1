# MBA-1: MarketPivot — Strategic Repositioning in the Age of AI Disruption

## Overview

Develop a **comprehensive 3-year strategic plan** for MediCore Health, a $2B healthcare IT company facing existential threat from AI-native startups and tech giants.

## Company Profile

| Metric | Value |
|---|---|
| Annual Revenue | $2B (declining 5% YoY) |
| EBITDA Margin | 18% |
| Cash Reserves | $500M |
| Debt | $800M (maturing in 3 years) |
| R&D Spend | 8% ($160M/year) |
| Customer Churn | 15% (up from 5%) |
| NPS Score | +12 (down from +45) |

## Revenue Breakdown

- **EHR Software**: 60% ($1.2B)
- **Medical Billing**: 25% ($500M)
- **Telehealth Platform**: 15% ($300M)

## Threats

1. **AI-native startups**: 1/10th the cost with AI-powered features
2. **Tech giants**: Google Health, Microsoft/Nuance, Amazon Clinic
3. **Regulatory changes**: CMS interoperability mandates commoditizing data moat

## Task

### 1. Strategic Options Analysis

Evaluate using weighted scoring model:

| Option | Description |
|---|---|
| **Build** | Develop proprietary AI capabilities in-house |
| **Buy** | Acquire AI healthcare startups |
| **Partner** | Form strategic alliances with tech giants |
| **Pivot** | Transform into a healthcare AI platform company |

### 2. Go-to-Market Transformation

- Enterprise license → SaaS/platform transition
- Customer retention strategy
- New customer acquisition
- AI-augmented product pricing

### 3. Financial Plan

- 3-year P&L projection
- Capital allocation (R&D, M&A, hiring)
- Debt management ($800M maturing)
- Break-even analysis for AI products

### 4. Organizational Transformation

- New roles/capabilities needed
- Talent acquisition plan
- Culture change (healthcare IT → AI-first)

### 5. Risk Analysis

- Key risks identification
- Risk mitigation plan
- Contingency plans (Plan B, Plan C)

## Deliverables

1. **Strategic Plan Document** (max 30 pages)
2. **Financial Model** (Excel with scenario analysis)
3. **Executive Presentation** (15 slides)
4. **Board Summary** (1 page)

## Frameworks to Consider

### Strategic Analysis
- SWOT analysis

- Blue Ocean Strategy

### Financial Modeling

- Scenario analysis (bull/base/bear)
